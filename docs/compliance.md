# Compliance

## Baseline rules

- Treat this repository as a reference index, not an authoritative legal publication
- Preserve source attribution and avoid transforming official meaning
- Start with metadata and links before any full-text storage
- Review privacy risk before storing fields that may identify natural persons

## MVP-safe fields

- law id and name
- article number and title
- case id, case number, court, decision date
- referenced article strings
- official URLs

## Deferred areas

- full-text mirroring
- editorial summaries
- AI-generated legal interpretation
